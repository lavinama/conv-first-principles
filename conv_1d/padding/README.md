We are going to pad in multiple dimensions.

### 1D Padding
**input:** [m, W], **padding:** [p], **output:** [m, W + 2p]

### 3D Padding
**input:** [m, W, H], **padding:** [p], **output:** [m, W + 2p, H + 2p]

### 4D Padding
**input:** [m, W, H, c], **padding:** [p], **output:** [m, W + 2p, H + 2p, c]
